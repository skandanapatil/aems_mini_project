# aems_mini_project

Windows user

1. to run the code we need a virtual env, as project level dependencies are added in virtual env
2. to create a virtual env - py -m venv venv (first time)
3. to activate the env - venv/Scripts/activate (run everytime, as they help you in activating your virtual env)
4. pip install -r requirements.txt - when ever new dependencies are added
5. to run the code py customer_churn.py
